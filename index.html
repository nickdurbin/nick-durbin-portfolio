Yelp Clone

This application utilizes HTML, CSS, Bootstrap, Ruby/Rails, and hosted on Heroku via SSH keys. The purpose of the application is for users to create and review coffee places. A user has the ability to create a profile, upload photos, leave comments, and give a rating.

Nomster follows CRUD and has a Google API to allow users to pinpoint their favorite local coffee shop. You can follow this link to look at the code on GitHub.

Movie Quote App

Weather App

This application was created with Apache and Cordova to take advantage of Javascript on the VSC Community platform. The application was built for mobile Androis devices and has the capability of geolocation with a Google API and an API with limited usage from AccuWeather.

A user can simply input a location or zip code to find the current weather conditions for that area or allow geotracking to get the weather for their current position. Here is a link to the code on GitHub.
